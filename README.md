# 프로젝트 2 
# [공정 조건별 품질 패턴 도출을 통한 제어 전략 제안]

> 성결대학교 주관 데이터사이언스 경진대회 / 2025년 4월 / 동상 수상

---

## 1. 프로젝트 개요
화장품 혼합탱크 공정 데이터 기반 품질 최적화 및 공정 제어 전략 수립 프로젝트

---

## 2. 프로젝트 배경
- 생산 공정 품질 안정화를 위한 최적 조건 탐색 필요성 증대
- 품질 지표와 공정 변수 간 연계 분석을 통한 데이터 기반 양호 패턴 식별
- 실질적 공정 제어 및 관리에 활용 가능한 객관적 표준 가이드 도출 지향

---

## 3. 분석 목표
- 품질을 대표할 수 있는 핵심 지표 선정
- 품질 그룹별 공정 변수 차이 비교
- 공정 변수 간 상관관계 분석
- 중복 제어를 줄이고 효율적인 제어 전략 제안

---

## 4. 사용 데이터
- Process Data : 혼합 공정 주요 변수 데이터  
- QC Data : 품질 측정 데이터  

---

## 🔒 Data Note
- 실제 화장품 제조 공정 데이터 기반 프로젝트 수행
- 기업 보안 및 민감 정보 보호를 위한 원본 데이터 비공개 처리
- 발표 자료 및 분석 결과 기반의 포트폴리오용 분석 코드 재구성
- 전체 분석 프레임워크 및 핵심 로직 구현 중심의 정리

---

## 5. 분석 흐름
1. Process Data와 QC Data 병합  
2. 품질 기준 변수로 `Weight` 선정  
3. 평균 기준으로 Good / Bad 그룹 분류  
4. `ReactA_Temp`, `ReactB_Temp` 등 주요 공정 변수 비교  
5. 공정 변수 간 상관관계 분석  
6. 품질 개선을 위한 제어 전략 도출  

---

## 6. 주요 분석 내용
- `Weight`를 품질 판단 기준으로 설정  
- Good / Bad 그룹 간 공정 변수 평균 비교  
- `ReactA_Temp`, `ReactB_Temp` 중심의 패턴 확인  
- 상관관계 히트맵을 통해 변수 간 연관성 파악  

---

## 7. 주요 인사이트
- 좋은 품질 그룹과 나쁜 품질 그룹은 일부 온도 변수에서 나타난 차이 
- 상관관계 분석 결과, 일부 공정 변수는 함께 움직이는 경향으로 중복 제어 가능성 확인
- 독립적 관리 가치가 높은 핵심 변수 식별을 통한 공정 제어 우선순위 수립
- 향후 머신러닝 기반 품질 예측 모델로 확장 가능한 기반 마련

---

## 8. 결과물
- 품질 그룹별 공정 변수 평균 비교 시각화  
- 공정 변수 간 상관관계 히트맵  
- 품질 제어 전략 제안  

---
