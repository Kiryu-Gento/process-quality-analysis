## 1. 프로젝트 개요
화장품 혼합탱크 공정 데이터 기반 품질 최적화 및 공정 제어 전략 수립 프로젝트

---

## 2. 프로젝트 배경
- **품질 관리의 중요성** : 제조 공정 내 불량률 감소, 고객 만족도 향상 및 비용 절감을 위한 핵심 요소
- **데이터 기반 제어 필요** : 숙련공의 경험이 아닌 객관적 공정 데이터 분석을 통한 생산 효율성 개선 요구
- **최적 조건 탐색** : 양호 품질 시의 공정 조건을 데이터로 도출하여 현장 활용 기준 마련 지향

---

## 3. 분석 목표
- **핵심 품질 지표 선정** : 실무적 중요도와 데이터 가용성을 고려한 최적 품질 KPI 수립
- **그룹 간 변수 비교** : 품질 등급별 공정 변수(온도, 전류 등)의 유의미한 패턴 차이 분석
- **상관관계 최적화** : 변수 간 의존성 파악을 통한 중복 제어 방지 및 독립 제어 변수 식별
- **제어 전략 수립** : 도출된 양호 패턴을 기반으로 한 실질적인 공정 관리 기준 제안

---

## 4. 사용 데이터
- **Process Data** : 혼합 공정 내 11개 센서(온도, 전류, 전력 등)로부터 수집된 시계열 데이터
- **QC Data** : 날짜별로 기록된 5종의 품질 지표(Weight, pH, HSO 등) 측정 데이터

## 5. 분석 흐름
1. **데이터 전처리** : 품질 지표 중 이상치를 제거하고 시계열 공정 데이터와 날짜별 병합 수행  
2. **품질 기준 선정** : 지표별 데이터 분포 검토 후 Weight를 핵심 품질 기준으로 확정  
3. **등급 분류** : Weight 평균값(775g)을 기준으로 Good / Bad 품질 그룹 세분화
4. **평균 비교 분석** : 각 그룹의 공정 변수 평균값 대조를 통한 품질 영향 변수 도출  
5. **상관관계 분석** : 히트맵을 통한 변수 간 연관성 파악 및 중복 제어 요소 필터링  
6. **제어 전략 도출** : 분석 결과를 바탕으로 최적 온도 구간 및 제어 우선순위 제안 

---

## 6. 주요 분석 내용

### 6-1. 품질 지표 수립
- **핵심 지표 선정** : 타 지표(pH 등) 대비 데이터 분포가 풍부하고 실무 중요도가 높은 Weight(충전량) 선정
- **품질 등급 정의** : Weight 평균 775g을 기점으로 이상(Good) 및 미만(Bad) 그룹으로 데이터 라벨링

👉 **데이터 가용성과 실무 중요도가 높은 Weight를 핵심 KPI로 선정하여 분석의 객관적 타당성 확보**

### 6-2. 공정 변수 영향도 분석
- **핵심 온도 변수 식별** : ReactA_Temp는 Good 그룹에서 평균 21.5도로 더 높게 나타나는 패턴 확인
- **불량 유발 요인 진단** : ReactB_Temp는 Bad 그룹에서 평균 29.5도로 높아 공정 과열 가능성 확인 

👉 **전류/전력 대비 온도 변수가 품질 판별에 결정적인 기여를 함을 정량적으로 증명**

### 6-3. 변수 간 상관성 검토
- **변수 동조화 파악** : ReactB, ReactD, ReactE 간의 매우 높은 상관관계(0.99) 확인 및 중복 제어 위험 식별
- **독립 제어 변수 도출** : 타 변수와 상관성이 낮은 ReactA_Temp를 단독 제어 가능한 핵심 변수로 분류

👉 **높은 상관관계를 갖는 변수군은 통합 관리하고, 독립 변수는 개별 정밀 제어하는 효율화 방안 수립**

---

## 7. 주요 인사이트
- **품질 등급별 분포 차이** : Good/Bad 그룹 간 특정 온도 변수에서 뚜렷한 데이터 편차 확인 
- **변수 간 상관성 파악** : 일부 변수의 동조화 경향에 따른 중복 제어 요소 및 간섭 가능성 식별
- **제어 효율화** : 독립 관리 변수 선정을 통한 공정 제어 우선순위 설정 및 운영 가이드 마련
- **모델 확장성** : 향후 머신러닝 기반 품질 예측 및 이상 탐지 모델로의 확장 기틀 마련
