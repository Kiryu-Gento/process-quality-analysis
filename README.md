# 공정 조건별 품질 패턴 도출을 통한 제어 전략 제안

> 성결대학교 주관 데이터사이언스 경진대회 / 2025년 4월 / 동상 수상

---

## 1. 프로젝트 개요

화장품 혼합탱크 공정 데이터 분석으로,  
좋은 품질이 나오는 공정 조건 도출를 이용한 제어 전략 제안 프로젝트

---

## 2. 프로젝트 배경
생산 공정에서는 품질이 안정적으로 유지되는 조건을 찾는 것이 중요합니다.  
본 프로젝트에서는 품질 지표와 공정 변수를 연결하여,  
좋은 품질이 나타나는 패턴을 데이터 기반으로 확인하고  
공정 제어에 활용할 수 있는 기준을 도출하고자 했습니다.

---

## 3. 분석 목표
- 품질을 대표할 수 있는 핵심 지표 선정
- 품질 그룹별 공정 변수 차이 비교
- 공정 변수 간 상관관계 분석
- 중복 제어를 줄이고 효율적인 제어 전략 제안

---

## 4. 사용 데이터
- Process Data: 혼합 공정 주요 변수 데이터  
- QC Data: 품질 측정 데이터  

---

## 🔒 Data Note
본 프로젝트는 실제 화장품 제조 공정 데이터를 기반으로 수행되었습니다.  
기업 데이터 특성상 민감 정보가 포함되어 있어 원본 데이터는 공개하지 않았습니다.  

따라서 현재 저장소의 코드는 발표 자료와 분석 결과를 기반으로 재구성한 포트폴리오용 코드이며,  
분석 흐름과 핵심 로직을 중심으로 정리하였습니다.

---

## 5. 분석 흐름
1. Process Data와 QC Data 병합  
2. 품질 기준 변수로 `Weight` 선정  
3. 평균 기준으로 Good / Bad 그룹 분류  
4. `ReactA_Temp`, `ReactB_Temp` 등 주요 공정 변수 비교  
5. 공정 변수 간 상관관계 분석  
6. 품질 개선을 위한 제어 전략 도출  

---

## 6. 주요 분석 내용
- `Weight`를 품질 판단 기준으로 설정  
- Good / Bad 그룹 간 공정 변수 평균 비교  
- `ReactA_Temp`, `ReactB_Temp` 중심의 패턴 확인  
- 상관관계 히트맵을 통해 변수 간 연관성 파악  

---

## 7. 주요 인사이트
- 좋은 품질 그룹과 나쁜 품질 그룹은 일부 온도 변수에서 차이를 보였다.  
- 상관관계 분석 결과, 일부 공정 변수는 함께 움직이는 경향이 있어 중복 제어 가능성이 확인되었다.  
- 반면 독립적으로 관리할 가치가 있는 변수도 존재하여 제어 우선순위 설정에 활용 가능했다.  
- 향후 머신러닝 기반 품질 예측 모델로 확장 가능한 기반을 마련했다.  

---

## 8. 결과물
- 품질 그룹별 공정 변수 평균 비교 시각화  
- 공정 변수 간 상관관계 히트맵  
- 품질 제어 전략 제안
